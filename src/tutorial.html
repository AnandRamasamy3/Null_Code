Null Code :; is an Open Source Programming Language, written in python.
Simple syntax is the way to write complex codes efficiently. Null Code provides simple syntax form to the code.

Variables

Variable declaration is very simple in Null Code. You don't need to type datatype of the variable when declaration. You can just type the variable name and it's value. The compiler automatically takes the value and finds its datatype.

a=3
b=1.1
c="hello wolrd"
d=true
f=false

Arithmetic operations done easily through direct commands. Operator priority is

. () / * % + - is lt le gt ge and or not
Example Code
a=1+2-(5*10)



Operators

There are some conditional operators to make good conditions.

a=5
b=10
a is b # checks whether a is equal to b
a lt b # checks whether a is less than b
a le b # checks whether a is less than or equal to b
a gt b # checks whether a is greater than b
a ge b # checks whether a is greater than or equal to b



Gates

There are three common logic gate operators used in this language too.

a=true and false # and gate
b=true or false # or gate
c=not false # not gate



If

If is the simple condition to execute some set of code. But in here there is no "else if" or "else". Syntax for if condition is very simple.

  statement

Break

Using break the parent loop can be stopped.

for variable_name from start_value to end_value:
 if condition:
    break
  ;
;

while condition_1:
 if condition_2:
    break
  ;
;



Continue

Using Continue the current position of the execution is stopped and the next iteration of the parent loop condition is executed.

for variable_name from start_value to end_value:
 if condition:
    continue
  ;
;

while condition_1:
 if condition_2:
    continue
  ;
;
